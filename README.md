# Primequiz 🧠

Primequiz is an AI-powered educational platform built on **GenLayer**, featuring "Intelligent Contracts" that generate study materials and quizzes dynamically. Users can learn about any topic or upload their own materials to test their knowledge.

## 🚀 Key Features

- **AI-Driven Quiz Generation**: Create custom quizzes on any topic using the power of GenLayer's non-deterministic AI execution.
- **Custom Material Analysis**: Upload your own study material, and the Intelligent Contract will analyze it to generate relevant questions.
- **On-Chain Grading**: Grading is performed deterministically within the smart contract, ensuring transparency and fairness.
- **Study Mode**: Access comprehensive study materials generated by AI before taking the quiz.
- **Responsive UI**: A modern, premium interface built with React, Tailwind CSS, and shadcn/ui.

## 🛠️ Tech Stack

- **Frontend**: [Vite](https://vitejs.dev/), [React](https://reactjs.org/), [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/), [shadcn/ui](https://ui.shadcn.com/)
- **Blockchain**: [GenLayer](https://genlayer.com/) (using `genlayer-js`)
- **State Management**: [TanStack Query](https://tanstack.com/query/latest)
- **Icons**: [Lucide React](https://lucide.dev/)

## 📜 Intelligent Contracts

The platform uses two main GenLayer Intelligent Contracts:

1.  **QuizContract (quiz1)**: 
    - Handles topic-based quiz generation.
    - Generates 10 multiple-choice questions and study material for a given topic and difficulty.
    - Manages user answer submission and results breakdown on-chain.
2.  **QuizGenerator (quiz2)**:
    - Specializes in generating quizzes from user-provided text/material.
    - Stores generated quizzes by ID for later retrieval.
    - Grades user answers against the stored quiz data on-chain.

## 🏁 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later)
- [npm](https://www.npmjs.com/) or [bun](https://bun.sh/)

### Installation

1.  **Clone the repository**:
    ```sh
    git clone <repository-url>
    cd Primequiz
    ```

2.  **Install dependencies**:
    ```sh
    npm install
    # OR
    bun install
    ```

3.  **Environment Setup**:
    Create a `.env.local` file in the root directory and add your GenLayer key:
    ```env
    VITE_GENLAYER_KEY=your_genlayer_api_key_here
    ```

4.  **Run the development server**:
    ```sh
    npm run dev
    ```

## 📂 Project Structure

- `/src/pages`: Contains the main application views (Landing, Quiz Setup, Results, etc.).
- `/src/components`: Reusable UI components (Question cards, progress indicators).
- `/src/hooks`: Custom React hooks for interacting with GenLayer and managing state.
- `quiz1 intelligent contract`: Python code for the topic-based GenLayer contract.
- `quiz2 intelligent contract`: Python code for the material-based GenLayer contract.
